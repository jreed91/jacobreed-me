How Verizon’s Advertising Header Works:

In short, Verizon is packaging and selling subscriber information, acting as a data broker on real-time advertising exchanges.

Essentially what Verizon is doing is injecting a unique identifier into each request you make. Since this being done at the application layer, there is no way for you to block it. The only thing that blocks this is by using SSL or VPN. You can check to see if you’re being injected here.
